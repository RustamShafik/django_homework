# Django SkyPro Homework Project (1)

## Описание проекта

Это учебный проект на Django, выполненный в рамках домашнего задания курса SkyPro по Python.  
В проекте реализованы базовые настройки Django, создание и регистрация приложения, маршрутизация, работа с шаблонами и подключение стилей Bootstrap.

---

## Структура проекта

- Создано виртуальное окружение (`venv`) для изоляции зависимостей.
- Инициализирован новый проект Django (`config`).
- В корне проекта добавлен `.gitignore`, настроен GitFlow:
  - Основные ветки: `main`, `develop`
  - Для выполнения домашних заданий создаются отдельные ветки.

- Создано приложение `catalog`, зарегистрированное в настройках проекта.
- Настроена маршрутизация:
  - `/` — главная страница (`home.html`)
  - `/contacts/` — страница контактов (`contacts.html`)

- Используется фреймворк Bootstrap для стилизации страниц.
- Статические файлы (`css/bootstrap.min.css`) подключены через Django `{% static %}`.

## Как запустить проект

1. Клонировать репозиторий.
2. Создать и активировать виртуальное окружение:

```bash
python -m venv venv
source venv/bin/activate  # для Linux/Mac
venv\Scripts\activate  # для Windows
```

3. Установить зависимости (если есть `requirements.txt`):

```bash
pip install -r requirements.txt
```

4. Запустить сервер разработки:

```bash
python manage.py runserver
```

5. Открыть в браузере:
- [http://127.0.0.1:8000/](http://127.0.0.1:8000/) — главная страница
- [http://127.0.0.1:8000/contacts/](http://127.0.0.1:8000/contacts/) — страница контактов
